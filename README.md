## About

Horizon is a mod launcher inspired by Modrinth, designed to make managing and playing modded Minecraft effortless. Browse, install, and launch — everything in one place, no barriers.

## Features

- **Mod browsing & install** — search and install mods straight from the launcher
- **Instance management** — separate, isolated profiles per modpack
- **One-click launch** — handles game versions, mod loaders, and assets for you
- **Clean, modern UI** — built to stay out of your way

## Tech Stack

- **Electron** — desktop shell
- **TypeScript** — strict mode
- **[@xmcl/core](https://github.com/Voxelum/minecraft-launcher-core-node)** `^2.15.1` — Minecraft launch core

## Getting Started

```bash
# Clone
git clone https://github.com/LuuLoHvH/horizon.git
cd horizon

# Install
npm install

# Run in dev
npm run dev

# Build
npm run build
```
